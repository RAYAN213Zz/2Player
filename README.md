# Versus Throw

## Jouer en local
1. Ouvrir `public/index.html` dans un navigateur (ou servir le dossier `public/` via un hébergeur statique).
2. Cliquer sur “Jouer”, glisser sur la balle pour la lancer vers le trou.

## Héberger (ex. InfinityFree)
- Uploade le contenu du dossier `public/` dans le dossier `htdocs` de ton hébergement.
- Accède à l’URL publique fournie par l’hébergeur : le jeu tourne côté client uniquement.
